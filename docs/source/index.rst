11| Welcome to the official AgriOS documentation
12| ============================================
13| 
14| AgriOS is an open-source platform designed to help farms and agribusinesses manage
15| operations across production, inventory, sales, purchasing, accounting, and more.
16| Whether you are a farmer looking to streamline daily operations, an integrator
17| deploying AgriOS for customers, or a developer extending the system, this
18| documentation is organized to get you productive quickly.
19| 
20| Who this documentation is for
21| ------------------------------
22| 
23| - Farmers and business users: step-by-step guides for common tasks and workflows.
24| - System administrators: deployment, hosting and database management guidance.
25| - Developers and module authors: design overview, development practices, and
26|   integration guidance.
27| - Partners and contributors: information about partnerships, distribution, and
28|   contribution processes.
29| 
30| How this documentation is organized
31| -----------------------------------
32| 
33| The content is grouped into a small set of primary sections so you can quickly
34| find what you need:
35| 
36| - User Guide — user-facing workflows and how-to articles for day-to-day use.
37| - Database Management — installation, hosting and versioning notes.
38| - Developer Guide
