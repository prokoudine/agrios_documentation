Inventory
=========

This page provides a walkthrough for users on how to manage inventory in AgriOS/Odoo.

Contents
--------

.. contents::
    :local:
    :depth: 2

Menu Structure
--------------

- Inventory
    - Products
    - Stock Moves
    - Locations
    - Transfers
    - Reports

Walkthrough
-----------

1. **Accessing Inventory Module**
   - Go to the "Inventory" menu from the dashboard.

2. **Managing Products**
   - Add, edit, or view products in inventory.

3. **Stock Moves and Transfers**
   - Record stock moves and manage transfers between locations.

4. **Location Management**
   - Organize and manage storage locations.

5. **Reporting**
   - Generate inventory reports and analytics.
