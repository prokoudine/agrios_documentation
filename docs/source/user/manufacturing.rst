Manufacturing
=============

This page provides a walkthrough for users on how to manage manufacturing processes in AgriOS/Odoo.

Contents
--------

.. contents::
    :local:
    :depth: 2

Menu Structure
--------------

- Manufacturing
    - Orders
    - Work Orders
    - Products
    - Bill of Materials
    - Reports

Walkthrough
-----------

1. **Accessing Manufacturing Module**
   - Go to the "Manufacturing" menu from the dashboard.

2. **Managing Manufacturing Orders**
   - Create and track manufacturing orders and work orders.

3. **Product and BOM Management**
   - Manage products and bill of materials.

4. **Reporting**
   - Generate manufacturing reports and analytics.
