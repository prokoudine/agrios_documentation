Farmers
=======

This page provides guidance for users on how to manage farmer profiles and related activities in AgriOS/Odoo.

Contents
--------

.. contents::
    :local:
    :depth: 2

Menu Structure
--------------

- Farmers
    - Profiles
    - Activities
    - Reports

Walkthrough
-----------

1. **Accessing Farmers Module**
   - Navigate to the "Farmers" menu from the dashboard.

2. **Managing Profiles**
   - Add, edit, or view farmer profiles and details.

3. **Tracking Activities**
   - Record and monitor farmer activities and engagement.

4. **Reporting**
   - Generate reports on farmer data and activities.
