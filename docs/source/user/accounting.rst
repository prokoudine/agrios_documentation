Accounting
==========

This page provides a walkthrough for users on how to manage accounting in AgriOS/Odoo.

Contents
--------

.. contents::
    :local:
    :depth: 2

Menu Structure
--------------

- Accounting
    - Invoices
    - Payments
    - Journals
    - Chart of Accounts
    - Reports

Walkthrough
-----------

1. **Accessing Accounting Module**
   - Go to the "Accounting" menu from the dashboard.

2. **Managing Invoices and Payments**
   - Create, view, and process invoices and payments.

3. **Journals and Chart of Accounts**
   - Manage journals and review the chart of accounts.

4. **Reporting**
   - Access financial reports and analytics.
