Sales
=====

This page provides a walkthrough for users on how to manage their sales in AgriOS/Odoo.

Contents
--------

.. contents::
    :local:
    :depth: 2

Menu Structure
--------------

- Sales
    - Quotations
    - Orders
    - Customers
    - Products
    - Invoices
    - Reports

Walkthrough
-----------

1. **Accessing Sales Module**
   - Navigate to the main dashboard and click on the "Sales" menu.

2. **Creating a Quotation**
   - Click on "Quotations".
   - Select "Create" to start a new quotation.
   - Fill in customer details, products, and pricing.
   - Save and send the quotation to the customer.

3. **Confirming Sales Orders**
   - Once a quotation is accepted, confirm it to create a sales order.
   - Review order details and validate.

4. **Managing Customers**
   - Use the "Customers" menu to add or edit customer information.

5. **Product Management**
   - Access "Products" to manage your product catalog, pricing, and inventory.

6. **Invoicing**
   - Generate invoices from confirmed sales orders.
   - Track payments and outstanding invoices.

7. **Reporting**
   - Use the "Reports" section to view sales performance, trends, and analytics.

For more detailed instructions, refer to the specific sections above or contact support for assistance.
